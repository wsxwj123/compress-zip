# BRIEF — compress-zip 需求定稿

## 背景与动机
现用的压缩 App 把中文名文件压成 zip 后，拿到 Windows 上解压文件名就乱码。想要一个 Mac 上访达右键就能用的压缩/解压工具，**头号目标是中文文件名跨平台不乱码**。

## 功能清单（输入 X → 得到 Y）
1. **右键压缩**：访达选中文件/文件夹（单个或多个）→ 右键"快捷操作 → 压缩"→ 弹窗选格式和加密方式 → 在源目录生成压缩包。**中文文件名在 Windows 上解压不乱码**。
   - **命名（改）**：单选用该项名字；**多选默认命名"归档.zip"**（不再用父目录名，避免"只选了部分却叫父目录名"的误导）。
   - **包内结构（改）**：**不加顶层文件夹前缀**（包里直接是所选项，对齐 macOS Finder；"套文件夹"由解压端按需加）。
   - **不写 macOS 元数据**：`.DS_Store` / `__MACOSX/` 不进包（否则污染包、干扰解压端的"单顶层项"判断）。
2. **右键解压**：访达选中压缩包 → 右键"快捷操作 → 解压"→ 解出，中文名不乱码。**解压位置默认=压缩包所在目录**。
   - **智能默认（改）**：包里只有 1 个顶层项 → 直接铺开；有多个顶层项 → 自动套一层以包名命名的文件夹（对齐 The Unarchiver / Keka，避免散文件污染当前目录、避免"归档/归档"套娃）。
   - **手选（保留）**：弹窗还提供"解压到指定目录""强制铺开""强制套文件夹"三个手动选项。
   - **同名改名（改）**：作用在**最终落地的顶层项**上（`名字` → `名字-1`），不是逐个文件改（否则文件夹被拆散）；自动改名、不弹窗。
3. **格式选择**：压缩时可选 zip（必备）/ 7z / tar.gz。
4. **加密可选**：压缩时可选①AES-256 强加密 ②传统 ZipCrypto 兼容加密 ③不加密；加密时输入密码。
5. **解压广覆盖**：能解常见格式（zip / 7z / tar.gz / rar 等，带密码 rar 不支持）。
6. **外部包兼容与安全（新增）**：
   - **Zip Slip 防护（安全红线）**：解压前校验每个成员路径不含 `..`、不以 `/` 开头，越界的拒绝——防止文件被写到目标目录外面覆盖系统文件。
   - **反斜杠归一化**：外部包路径用 `\` 的，解压时转 `/` 再建目录，避免解出带反斜杠的怪文件名。
   - 外部"带壳包"（顶层就一个文件夹）走智能默认自动铺开，符合直觉。

## 明确不做
- 不做 GUI 界面 App（走命令行内核 + 系统快捷操作）
- 不做 RAR 创建（专利，无免费库；RAR 只支持解压）
- 不联网、不做云端、不上传任何数据
- **压缩/解压都不删除、不覆盖原文件**；输出同名时自动改名避让（如 `名字-1.zip`）

## 成功标准
1. ★**中文名文件压成 zip，在 Windows 解压文件名不乱码**（头号，可自动验证：包内文件名为 UTF-8 字节且 UTF-8 标志位=bit11 置位）
2. 访达右键能触发压缩/解压，普通用户按教程能装上
3. 加密包能用密码正确解开，密码错误有明确提示
4. 单个和多个文件/文件夹混选都能正确打包（多选默认名"归档.zip"、不加顶层前缀）
5. 解压：智能默认（单项铺开/多项套壳）正确，手选三项可用，同名改名作用在顶层项、不覆盖已有文件
6. Zip Slip 恶意包解压被拒绝、目标目录外零文件落地（安全红线）

## 项目类型与平台范围
- 类型：**代码/脚本**（命令行压缩解压内核）+ **系统集成**（macOS 快捷操作/Automator 接入访达右键）
- 开发与运行平台：macOS 15.7.3
- 跨平台验证点：**Windows 解压不乱码**——本机无 Windows，用"检查 zip 包内文件名字节编码 + UTF-8 标志位"自动验证，等价于保证 Windows 不乱码

## 测试策略
- 核心内核（代码/脚本）：pytest 调用压缩/解压对外接口真实跑
- 头号验收（可自动化，无需真 Windows）：压缩后解析生成的 zip，断言中文文件名以 UTF-8 存储且 general purpose bit 11 置位；再用 Python 标准库以 cp437/gbk 反解验证不乱码路径
- 加密：压缩后用正确/错误密码解压，验证成功/失败行为
- 边界：中文名、名字带空格/emoji、多文件混选、大文件、同名冲突、空文件夹
- 快捷操作接入访达：靠用户在自己机器上实测（07 收尾）

## 敏感面声明（安全审计依据）
1. **碰哪些文件**：读用户选中的文件/文件夹；写压缩包到源目录；解压写文件到用户选的目标路径。**绝不删除、绝不覆盖原文件**（同名改名避让）。
2. **密钥/密码**：加密密码由用户实时输入，**绝不落盘、绝不写日志、绝不明文留存**，用完即弃。
3. **外发数据**：无。纯本地运行，不联网、不外发任何内容。
